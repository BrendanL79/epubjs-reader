Epub.js Reader
================================

[Try it while reading Moby Dick](http://epubjs-reader.appspot.com/)

Getting Started
-------------------------

Rendering is handled by [Epub.js](http://futurepress.github.com/epub.js/)

Deploying to App Engine
-------------------------

Update the app.yaml with you application id.

Then run:
```
appcfg.py update .
```
